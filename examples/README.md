# Sample Charts

These SVG charts are generated by GitHub Star Tracker. By default they adapt to dark and light mode via `@media (prefers-color-scheme: dark)`; you can force a palette with `chart-theme: light|dark`.

Toggle your system or browser theme to see the effect.

Chart appearance is configurable — smoothing, point markers, animation, milestone lines (built-in or custom), the Y-axis floor, the time window, and a moving-average trend line. See **[Star Trend Charts](../docs/wiki/Star-Trend-Charts.md)** for the full list.
