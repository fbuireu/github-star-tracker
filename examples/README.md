# Sample Charts

These SVG charts are generated by GitHub Star Tracker. By default they adapt to dark and light mode via `@media (prefers-color-scheme: dark)`; you can force a palette with `chart-theme: light|dark`. Toggle your system or browser theme to see the effect.

This page is the visual reference for every chart option. Each comparison below uses the same data so only the option being demonstrated changes. See **[Star Trend Charts](../docs/wiki/Star-Trend-Charts.md)** for the prose docs.

## Curve Styles (`chart-curve`)

The curve used to connect points when `chart-smoothing` is on. All four use the same stepped data so the differences are clear: `monotone` (default), `cubic-bezier` and `rounded-step` keep the plateaus flat; only `catmull-rom` overshoots at the foot of a step, dipping below the previous value.

| `monotone` (default) | `catmull-rom` |
|---|---|
| ![monotone](curve-monotone.svg) | ![catmull-rom](curve-catmull-rom.svg) |

| `cubic-bezier` | `rounded-step` |
|---|---|
| ![cubic-bezier](curve-cubic-bezier.svg) | ![rounded-step](curve-rounded-step.svg) |

> Email charts (QuickChart) approximate these: `monotone` is exact, `rounded-step` falls back to `monotone`, and `catmull-rom` and `cubic-bezier` both render as a tensioned spline.
